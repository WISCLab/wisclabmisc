# wisclabmisc

The wisclabmisc package provides standardized analysis code from [WISC
Lab project](https://kidspeech.wisc.edu/) projects. The package also
collects useful data-cleaning function and reference datasets for our
lab. See the package documentation website
<https://wisclab.github.io/wisclabmisc/>.

Analysis code for selected publications is included on the [articles
page](https://wisclab.github.io/wisclabmisc/articles/index.html).

## Installation

Install wisclabmisc from
[GitHub](https://github.com/WISCLab/wisclabmisc) with:

``` r

# install.packages("devtools")
devtools::install_github("wisclab/wisclabmisc")
```
